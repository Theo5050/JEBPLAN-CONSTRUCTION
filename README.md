Official website for JEBPLAN Construction - Your trusted partner for scaffolding solutions, security doors, and construction accessories in Accra, Ghana.

 About
Founded in 2020 by Justice Essoun Baah, JEBPLAN Construction has grown to become a trusted name in Ghana's construction industry. This website serves as our online presence, product catalog, and payment portal.

 Key Features
- Product Catalog: Scaffolding, doors, bathroom products with pricing
- Payment Integration: Secure payments via Paystack (MTN MoMo, Cards, Bank Transfer)
- Admin Dashboard: View contact submissions and payment records
- Certification Gallery: 23 professional certificates
- Responsive Design: Works perfectly on mobile, tablet, and desktop
- Auto-updating Copyright: Always shows current year

 Built With
- Frontend: HTML5, CSS3, JavaScript
- Backend: Node.js, Express.js
- Database: MongoDB Atlas
- Payment: Paystack API
- Hosting: Render
- version Control, Git & GitHub

  Installation

Prerequisites
- Node.js (v14 or higher)
- MongoDB Atlas account
- Paystack account (for payments)

 Steps
1. Clone the repository**
   ```bash
   git clone https://github.com/Theo5050/JEBPLAN-CONSTRUCTION.git
   cd JEBPLAN-CONSTRUCTION
Install dependencies

bash
npm install
Set up environment variables
Create a .env file in the root directory:

env
MONGODB_URI=your_mongodb_atlas_uri
PAYSTACK_SECRET_KEY=your_paystack_secret_key
JWT_SECRET=your_jwt_secret
PORT=5000
Run the development server

bash
npm run dev
Open your browser
Navigate to http://localhost:5000
